使用 KOReader 阅读

KOReader 可以使用 OPDS v1 访问 Komga。

阅读进度同步​

阅读进度的同步方式如下：

  • 对于预分页的书籍，阅读进度将按页准确显示。
  • 对于常规的 EPUB 书籍，KOReader 将会在章节开头记录阅读进度，无法记录章节内的进度。 在 Komga 和 KOReader 之间切换时，章节内的阅读进度会丢失。

配置​

要使用使用 KOReader 同步和 Komga，Komga 用户必须有 KOREADER_SYNC 角色。

你还需要启用计算 KOReader 文件的哈希值。

生成 API 密钥​

KOReader 同步 Komga API 使用了一套专用端点，需要 API 密钥来访问。 用户可以在 账户设置 页面生成 API 密钥。 务必记下生成的密钥，因为它以后不会再次显示。

如果您有多个 KOReader 设备，建议对每个设备使用不同的 API 密钥。

设置 KOReader​

自定义同步服务器​

  1. 在 KOReader 菜单中，转到 进度同步。

  2. 点击 自定义同步服务器。

  3. 输入你的 Komga 服务器地址，然后是 /koreader。 例如：https://demo.komga.org/koreader

  4. 点击 注册 / 登录。

  5. 在 注册 / 登录 对话框中，输入以下内容：

    • username：Komga API 密钥
    • password：任意虚拟值，这个字段不被 Komga 使用

  6. 点击 登录。

文档匹配方法​

  1. 在 KOReader 菜单中，转到 进度同步。

  2. 点击 文档匹配方法。

  3. 选择 二进制匹配。

问题排查​

Komga 日志​

通过把 logging.level.org.springframework.web.filter.CommonsRequestLoggingFilter 设置为 DEBUG，Komga 可以记录所有传入的 HTTP 请求，包括请求头和请求体。

你还可以通过 Komga 服务器的 /actuator/httpexchanges 端点查看最近的 HTTP 交互记录。

值得注意的社区插件​

Kobo Sync​

如果你在 Kobo 设备上使用 KOReader，你可以通过使用 kobo.koplugin，直接在 KOReader 中设置 Kobo Sync 并打开书籍。

ComicReader​

ComicReader：一个提升漫画阅读体验的插件。

ComicMeta​

ComicMeta：一个从你的漫画中提取 ComicInfo 元数据的插件。

Komga Bulk Downloader​

Komga Bulk Downloader：一个允许你批量下载书籍的插件，从而使你的书籍更容易离线阅读。
