# Read with MangaBox

[MangaBox](https://zpaolo11x.github.io/mangabox/) is a streamlined, multiplatform Komga client. It's free and open-source, available for Windows, macOS, Linux, iOS/ipadOS and Android. It is based on web technologies so you can also deploy it as a web client or PWA.

<img src="/assets/media/guides/mangabox/mangabox.png" style={{maxHeight: '500px'}}/>

## Features

- Connect easily to your Komga server from the onboarding screen
- Light and dark themes with custom color highlights
- Touch and gesture support for tablets and phones
- Beautiful two-pages reading with paper-style visuals, page-turn animations and thumbnail previews
- Sync reading progress with the server
- Download books for offline reading
- Quick filter libraries by language, read status, and direction
- Global search for series and volumes

## Track read progress

MangaBox can sync your reading progress with the server. When you read a book offline, read progress is synced back once the server is reachable again.
