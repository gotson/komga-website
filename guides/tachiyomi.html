# Read with Tachiyomi

# Install and configure

Komga has an official extension for Tachiyomi (opens new window), available from within Tachiyomi's extension menu.

Once installed, the extension will let you configure up to 3 Komga servers.

You will need to configure the extension with your server address, username and password before you can use it.

TIP

Example server address: http://my.server:8080(/baseUrl)

Note that there is no trailing slash (/).

# Browse

You can browse series, use the filter menu, and search for series.

# Track read progress

Tachiyomi starting from 0.12.1 can keep your read progress in sync with your Komga server. Check the official Tachiyomi website for instructions (opens new window).

Requirements:

  • Tachiyomi 0.12.1+
  • Komga 0. 95.2+
  • Tachiyomi Komga extension 1.2.24+

# Changelog and Compatibility

Each version of the Komga extension need a specific version of the Komga server to work properly. Check the CHANGELOG (opens new window) for more information.
